# Content Model

## Languages

Public content is available in:

- English: `en`
- Traditional Chinese: `zh-TW`
- Simplified Chinese: `zh-CN`

All three public language fields are required for publishable text content.

## Time Rules

- Admin scheduling fields use Taiwan time in the UI.
- Stored timestamps use UTC.
- Tour dates use the concert location timezone for display and UTC for sorting.

## Empty States

- News: hidden when inactive or missing.
- Tour: "Tour dates coming soon".
- Album mosaic: "Album covers coming soon".
- Videos: "Videos coming soon".
- Music: "Music links coming soon".
- Merch: "Merch coming soon" unless an external URL is configured.
- Social footer: hidden when no links exist.

## Media

Allowed upload formats:

- PNG
- JPG/JPEG
- WebP

SVG upload is intentionally excluded for version 1.
